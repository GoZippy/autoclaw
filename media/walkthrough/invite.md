### Invite a single agent

The guided wizard walks you through it:

1. **Tool** — picks the join lane (MCP, HTTP bridge, filesystem, or native /loop).
2. **Role** — what it works on (shown on the board).
3. **Behavioral type** — suggested from the role; take it or change it.
4. **Scope** — optional path globs it may touch.

It mints a single-use invite token and copies a ready-to-paste join prompt for that exact tool.
