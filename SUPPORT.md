# Support AutoClaw 💚

AutoClaw is built and maintained by **Zippy Technologies LLC** — a tiny team
(mostly one developer) trying to make autonomous AI coding agents that actually
help. **Personal and educational use is free, forever.** If AutoClaw saves you
time, here are the ways you can help keep it alive and growing.

> All of these are optional. Nothing in AutoClaw is locked behind a paywall for
> individual use. Support is goodwill — and it directly funds the bills and the
> time to keep building.

---

## ⭐ Leave a review (free, helps the most)

Reviews drive installs, which grow the project. Two minutes, costs nothing:

- **VS Code Marketplace:** https://marketplace.visualstudio.com/items?itemName=ZippyTechnologiesLLC.autoclaw&ssr=false#review-details
- **Open VSX:** https://open-vsx.org/extension/ZippyTechnologiesLLC/autoclaw/reviews

You can also rate it right inside the extension: run **"AutoClaw: Support /
Donate"** from the command palette.
