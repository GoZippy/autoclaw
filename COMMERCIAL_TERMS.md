# AutoClaw Commercial Terms Summary

This is a plain-language summary, **not** a replacement for [LICENSE](LICENSE),
which is the authoritative legal source of truth. If anything here conflicts with
LICENSE, LICENSE wins.

- Free Community mode is available for **personal, educational, open-source, and
  evaluation use**.
- **Commercial use requires a paid license** (Solo / Pro / Teams / Enterprise).
- Solo/Pro licenses may be sold as **one-time major-version licenses**.
- A paid one-time license lets you use the **purchased major version forever**.
- A paid one-time license includes **updates for the stated update window**,
  usually 12 months.
- After the update window ends, you **keep using the last eligible version** — the
  license does not stop working.
- **Teams/Enterprise** features may require an active subscription or annual
  license.
- **Hosted services** (features that run on our servers) may require an active
  entitlement **or** a bring-your-own (BYO) API key.
- The **7-day Pro trial** grants local Pro features for evaluation; it does **not**
  grant commercial-use rights and does **not** unlock hosted (cost-to-us)
  services.

See [docs/licensing.md](docs/licensing.md) for the full model.
